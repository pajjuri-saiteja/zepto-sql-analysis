# zepto-sql-analysis# Zepto E-Commerce Data Analysis (SQL + PostgreSQL)

## 📌 Project Overview

This project analyzes Zepto's e-commerce product dataset using SQL to derive meaningful business insights. The analysis focuses on data cleaning, discount patterns, inventory availability, category performance, and revenue estimation.

The objective is to demonstrate practical SQL skills used in real-world retail and e-commerce analytics.

---

## 🚀 Tech Stack

* **SQL (PostgreSQL)**
* **CSV Dataset**
* **Jupyter Notebook (for exploration)**
* **Git & GitHub**

---

## 📂 Dataset Information

**Dataset:** `zepto_v2.csv`

The dataset contains information about:

* Product Name
* Category
* MRP (Maximum Retail Price)
* Discount Percentage
* Selling Price
* Available Quantity
* Weight
* Stock Availability

## 🧹 Data Cleaning Steps

The following preprocessing tasks were performed:

* Removed duplicate records.
* Standardized column names.
* Converted price values to consistent units.
* Handled missing or invalid entries.
* Verified stock availability indicators.
* Validated category information.

## 🔍 Business Questions Solved

### 1. Top 10 products with the highest discounts

Identified products offering maximum discounts to understand promotional strategies.

### 2. High-value products that are out of stock

Detected premium products requiring inventory replenishment.

### 3. Category-wise estimated revenue

Calculated potential revenue generated by each product category.

### 4. Premium products with low discount percentages

Analyzed high-end products maintaining strong profit margins.

### 5. Duplicate product analysis

Investigated products appearing across multiple SKUs.

### 6. Stock availability insights

Examined inventory distribution across categories.

## 🛠️ How to Run This Project

### Step 1: Create a PostgreSQL Database

```sql
CREATE DATABASE zepto_analysis;
```

### Step 2: Import the Dataset

Load `zepto_v2.csv` into PostgreSQL.

### Step 3: Execute the SQL Script

Run:

```text
sql/Zepto E-Commerce SQL Data Analysis.sql
```

### Step 4: Analyze the Results

Execute individual queries to generate business insights and reports.
